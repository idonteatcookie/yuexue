# yuexue-frontend 约学前端项目

## 技术栈
## Vue2 vue-router Webpack axios sass MintUI Iconfont

## 项目结构
```c
├── build                      // 构建相关  
├── config                     // 配置相关
├── src                        // 源代码
│   ├── api                    // 所有请求
│   ├── assets                 // 图片静态资源
│   ├── components             // 全局公用组件
│   ├── router                 // 路由
│   ├── styles                 // 全局样式
│   ├── utils                  // 全局公用方法
│   ├── views                  // 页面
│   ├── App.vue                // 入口页面
│   └── main.js                // 入口 加载组件 初始化等
├── static                     // 第三方不打包资源
├── .babelrc                   // babel-loader 配置
├── eslintrc.js                // eslint 配置项
├── .gitignore                 // git 忽略项
├── index.html                 // html 模板
└── package.json               // package.json
```
